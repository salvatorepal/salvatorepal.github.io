Creating a personal website on GitHub

Steps in order to create a website

  1. Create a GitHub account.
  2. Once you've finished creating your account, go ahead and select the repositories tab.
  3. Click on the green button that says "New".
  4. Now you will be able to create a name for your website as well as leaving a description.
  5. Now give your website a name and click "Create Repository" at the bottom.
  6. Here you will be decide between creating a new file on GitHub or uploading a file directly.
    • For this tutorial we will be uploading a file.
  7. Now, simply drag over the html & css file(s) you wish to upload and click "Commit Changes" on the bottom.
